# 中文稿去 AI 痕迹 reference

> 中文论文写作/润色时必读。配套 `chinese-paper.md`。范例用"膳食农药暴露评估"主题仅作示意，
> 实际写作替换为你项目的研究内容。可叠加 `humanizer-zh` 技能做进一步清洗。

1. AIGC 检测原理（困惑度 / 突发性）

## 1 AIGC 检测原理

| 指标 | 含义 | AI 特征 | 人类特征 | 对策 |
|------|------|---------|----------|------|
| 困惑度 Perplexity | 文本可预测性 | 低：高频安全词 | 高：低频词、个性化 | 注入专业术语、具体数据、案例 |
| 突发性 Burstiness | 句式节奏变化 | 低：句长均匀 15–25字 | 高：长短交替 | 刻意变化句长（8字短句↔60字长句） |

核心规则：写作时同时提高困惑度和突发性。简单同义词替换已失效，须从结构层面重构。

## 2 绝对禁止的模式

| 模式 | 示例 | 修正 |
|------|------|------|
| 填充动词 | "进行了系统性的分析/研究" | 删"进行了…的"，直接"分析了" |
| 大词堆砌 | "具有重要的公共卫生意义" | 全文 ≤2 次 |
| 段首重复 | 连续3段"本研究"开头 | 变换主语：数据显示/结果表明/分析发现 |
| 三段式法则 | 优势三条、局限三条、建议三条 | 条目数反映实际内容，非固定三条 |
| 过度对冲 | "在一定程度上""从某种意义上" | 直接给结论 |
| 镜像结果 | 讨论首段重复所有结果数字 | 讨论概述发现，不复制数据 |
| 虚假精确 | "约为 1.82341 倍" | "约为 1.8 倍" |
| 情感词汇 | "令人震惊的是""令人欣慰" | 删除，学术不用情感词 |
| 模糊归因 | "多位学者指出""已有研究表明" | 具体引用"Chen等[3]发现…" |
| 宣传语气 | "丰富的""显著优势""先进的" | 删除评价性形容词 |

### 2.1 学术 register（标题与措辞）

| 错 | 对 |
|----|----|
| 疑问标题"为何…不宜…" | 名词短语；禁"为何/如何/能否/是否/吗/？" |
| 口语绝对词"无一/没有任何/全都" | "均未/未见" |
| aims"目标有三：其一…" | 一句话连用动词陈述 |
| 文言序数"其一/其二/其三" | "第一/第二/第三" |
| 自指作主语"本综述发现…" | "本研究"（节制）或发现引导句 |
| 元叙述/设问"下面将讨论…""那么…？" | 删 |

自检：`^#+.*(为何|如何|能否|是否|吗|？)` 与 `无一|没有任何|其一.*其二` 命中即改。

## 3 套话黑名单

**一级禁用（出现即删）**：值得注意的是 / 不容忽视 / 不可或缺 / 至关重要 / 举足轻重 / 日益严峻 /
引发了广泛关注 / 具有深远影响 / 充分证明 / 有力证实 / 深刻揭示 / 深入探讨 / 以期。

**二级限用（全文各 ≤1 次）**：综上所述（仅讨论末段）/ 总之（仅结论段）/ 近年来（改具体年份）/
"随着……的不断提高/发展"（改具体事件）。同一段内"此外/同时/并且/另外"合计 ≤2 个。

**三级结构禁令**：
- "首先……其次……最后……"过度结构化 → 用因果或对比结构。
- 连续分号排比 → 改独立句子、长短交替。
- "(1)…(2)…(3)…"全文 ≤2 处 → 用段落自然展开。

## 4 提高困惑度与突发性

**困惑度**：低频专业术语替代通用词；用数据替代泛陈述（"检出率较低"→"检出率0.71%"）；
用具体文献替代"已有研究"；加研究地区独有信息；加方法学具体参数（迭代次数、阈值）。

**突发性**：每段最短句 ≤15 字、最长句 ≥40 字；连续段落开头词不重复（相邻3段首2字不同）；
每段连接词 ≤2 个；每3句至少1句非"主谓宾"直线结构；段落间字数差 ≥30%。

写作风格：长句后插短句打破节奏；同段 ≥3 种不同主语；逻辑用因果/转折/对比连接而非"此外""同时"
堆砌；适当保留科学不确定性（"目前数据尚不足以确定"）而非"充分证明"。

## 5 终稿语言去工程化（CRITICAL）

终稿是研究者第一作者视角下"我做了什么研究"，不是"我经历了哪些步骤"。下列字眼出现即未完成：

| 禁用过程语言 | 改写为 |
|--------------|--------|
| 修正版/修正后/旧版/之前的版本 | 直接说"本研究"，不暴露版本演进 |
| 4 种方法对比/比较了 N 种方法 | "主分析采用 X，敏感性分析采用 Y、Z" |
| 在数据审查阶段发现并修正了… | 不出现；问题修正属研究内部 |
| "因不代表一般人群暴露故不用…" | 不保留"为什么不用"的解释，直接写采用了什么 |

统计方法节按发表范例写：未检出值处理/统计分析各单段，**禁止**列"N 种方法+各自适用+各自局限"，
**禁止**写 R 包版本号、随机种子、PNG 渲染引擎、矢量化效率等工程细节。

**最小修改原则**（修订已认可章节）：按用户标注点字面执行，不趁机重写整章；其余不动；改完 grep 验证禁用词为 0。

## 6 GOOD/BAD 范例 + grep 自检正则（按部件）

> 范例主题为示意。每部件给 GOOD、BAD、可直接跑的 grep 正则。

### 6.1 摘要
GOOD：四段结构、【结果】含具体数字与 CI、第三人称过去时、不引用文献。
BAD：`【目的】<复述题名>` / `【方法】本研究进行了系统性的…研究` / `【结果】结果表明较高，存在一定风险` / `【结论】综上所述…具有重要意义[1]`。
```
进行了.{0,6}(研究|分析|评估|调查)               # 填充短语
【(目的|方法|结果|结论)】.*\[[\d,]+\]            # 摘要中引用(禁止)
具有.{0,4}(重要|重大|深远).{0,4}(意义|价值|影响)  # 大词空话
```

### 6.2 引言
GOOD：第1段含具体流行/政策数据 + `[待补充引用]`；第4段"本研究利用…数据，采用…方法，对…进行评估"。
BAD：`食品安全是全球重要的公共卫生问题。近年来，随着人们健康意识的不断提高…日益受到广泛关注…不可或缺…举足轻重`（万能开头、零数据零引用）。
```
^.{0,5}(食品安全|公共卫生|人类健康).{0,10}(重要|重大|严峻)  # 万能开头
日益.{0,4}(受到|引起|得到).{0,6}(关注|重视)                 # AI堆砌语
不可或缺|举足轻重|至关重要                                    # 冗余修饰
^本(研究|文)                                                  # 段首重复(>2段)
```

### 6.3 方法
GOOD：含公式、具体参数（迭代次数、阈值、加工因子）、软件版本，可重复；**只写做了什么**。
BAD-1：`本研究采用先进的…方法…进行了全面系统的评估…具有科学性强、准确度高等显著优势`（自吹、无参数）。
BAD-2（最常见）：`照护负担涉及多个相互关联变量，若直接同时纳入回归易产生多重共线性。MCA 是适用于多个
分类变量的降维方法，将类别投影到低维空间……正式分析前作数据质量核查（配对完整性、逐行校验、缺失核查）。
就医方式因与三项几乎不相关、构念不同，故未纳入合成。`（动机辩解 + 教科书定义 + 数据核查过程 + 排除理由，
全是不该进方法的"为什么/过程"——见 chinese-paper.md「方法只写做了什么」）。
GOOD 对照：`采用 MCA 对 12 个分类变量降维，取前三维度（累计解释 44.0%），命名为……；以维度得分为自变量、
ZBI 为因变量作线性回归。就医方式作现状单独描述。`
```
(先进|领先|创新性|科学性强|准确度高|显著优势)   # 自吹形容词
(原因是|因为?|为了|为避免|鉴于|考虑到).{0,12}(共线性|偏态|稳定|方差|可解释)  # 方法里的动机辩解→删
是一种?.{0,8}(方法|技术|模型)[，,].{0,20}(将|把|通过)                        # 教科书定义→删
(正式)?分析前.{0,10}(核查|检查|清洗)|合成前.{0,8}核查                         # 分析前过程流水账→删
(未纳入|不纳入|排除).{0,12}(因|由于|是因为)                                   # 排除理由→只说处置不说为何
```

### 6.4 结果
GOOD：纯客观、每句有数字、引用表图、精度统一。
BAD：`结果令人震惊地显示…远远超过了安全阈值…具有极其重要的警示意义…这可能是因为…`（情感词、无数字、评价语、推测）。
```
令人(震惊|担忧|欣慰|鼓舞|关注)                  # 情感词(结果节禁止)
(可能是因为|这表明|这提示|说明了|反映了|原因在于) # 解释性语言(留给讨论)
(远远|大大|显著)(超过|高于|低于)(?!\d)            # 模糊表达(应有具体数字)
(重要|关键|核心).{0,4}(意义|价值|发现|启示)      # 评价语(结果节禁止)
```

### 6.5 讨论
GOOD：`Chen等[N]采用…框架对…评估，各组均低于阈值。本研究…高于…，差异可能与…有关`（具体引用+解释）。
BAD：`与部分既往研究一致，同时也存在一些差异。一些研究发现了类似结果，另一些则得出不同结论…`（不引用、模糊归因、零信息量）。
```
值得注意的是                                     # 全文≤1次
综上所述                                         # 全文≤1次
在一定程度上|从某种意义上|从某个角度             # 过度对冲
(一些研究|部分学者|已有研究|有学者)(发现|认为|指出|报道)  # 模糊归因
```

### 6.6 参考文献
GOOD：`[2] Chen G, Cao Y, Li X, et al. Title[J]. Journal, 2024, 183: 114276.`（类型标识+卷期页全）。
BAD：`[1] 张三. 农药残留研究. 某期刊, 2020.`（缺[J]缺卷期页）/ 裸 URL / 中文"等"前作者不足。
```
\[\d+\]\s+[^\[]*[^]\]]\.\s*$        # 缺文献类型标识[J]/[M]…
^\[?\d+\]?\s*(https?://|www\.)       # 裸URL
[一-鿿]{2,3},\s*等\.         # 中文文献"等"前作者不足
```

### 6.7 全文连接词/结构机械感
```
(此外|同时|并且|另外|而且|与此同时|不仅如此)   # 同段合计≤2个
首先.{5,50}其次.{5,50}(最后|再次)               # 全文≤2处
[^。！？\n]*；[^。！？\n]*；[^。！？\n]*；         # 分号排比(同句≥3分号)
```

交付前对全文跑一遍上述正则，逐条命中必须改到 0（或解释为何保留）。
