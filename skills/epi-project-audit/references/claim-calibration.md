# Claim Calibration

用这份规则判断“结论是否写过头了”。

## 1. 可以相对直接写的情况

仅当以下条件同时满足时，才允许较强表述：
- 方向一致
- 主结果 CI 不跨无效线
- 敏感性分析不改方向
- 异质性低或可解释
- 质量评价不差
- 没有明显重叠/偏倚红旗

可用表述：
- “方向一致”
- “结果较稳”
- “支持……相关”
- “提示……风险增加/降低”

## 2. 必须收敛的情况

出现以下任一情况时，必须降级措辞：
- 单篇研究或单篇综述
- 预测区间跨无效线
- 高异质性
- 高重叠风险
- 叙述性证据
- GRADE 为 Low/Very Low
- 质量评价多数为 Low
- 研究组补充推断的证据等级

此时优先使用：
- “提示可能存在……”
- “方向上呈……”
- “支持……方向，但需审慎解释”
- “更适合视为方向性信号，而非确定性结论”

## 3. 常见过强表述替换

- “证实” → “提示” / “支持”
- “明确表明” → “显示……方向”
- “稳定保护作用” → “保护方向信号”
- “可以得出结论” → “当前证据支持……但需结合限制解释”
- “独立研究一致” → “方向一致，但仍存在概念重叠/来源重叠可能”

## 4. 审稿人最容易挑战的地方

优先检查：
- 因果语言是否超过研究设计
- 观察性与MR/试验结果冲突时是否过度偏向单方
- 主文是否忽略 prediction interval、overlap risk、GRADE 低等级
- 结果摘要是否把辅助性/叙述性证据写成核心定量结论
