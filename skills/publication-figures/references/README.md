# publication-figures / references

随 skill 落地的可视化参考资源（路径稳定，不依赖外部库位置）。提炼自用户私有 R 可视化库
（50 套常用 + 130 套进阶"优雅"配方，2023 合集）。

## 文件
- **`chart-gallery.md`** —— 图类型选型与开创思路画廊。出图前先读：按"分析意图"选最达意图型 +
  优雅升级建议（云雨/山脊/哑铃/桑基/旭日/funkyheatmap…）+ "诉求→升级"速查表。SKILL.md §0 工作流入口。
- **`recipes_common_50/`** —— 50 类常用生物统计图的 R 配方代码（柱/箱/小提琴/热图/火山/venn/UpSet/
  雷达/桑基/棒棒糖/富集圈图/生存曲线/森林/列线图/ROC/PCA/瀑布等）。文件名即类目（如 `40ROC曲线.R`）。

## 重要：配方是"技法起点"，不是发表级成品
`recipes_common_50/` 来自教程脚本，普遍存在：base `pdf()` 设备（非 cairo_pdf）、硬编码 `setwd("D:\\...")`、
英寸默认尺寸、无字体嵌入、无中文字体注册——**全部不达本 skill 发表标准**。用法：只参考"这种图用什么包/
怎么搭几何对象/怎么标显著性"，**最终出图一律按 SKILL.md §1–§12 重写**（mm 尺寸、字体嵌入、ggsci 配色、
PDF+300DPI PNG 双存、Read 验证）。

## 进阶/优雅库（130 套 → 已收录 127 个 .R 代码）
- **`recipes_advanced/<文件夹名>/work.R|plot.R`** —— 127 个进阶配方代码（热图族/云雨/山脊/哑铃/桑基/
  ggraph 网络/ggtree 树/circlize 基因组/南丁格尔/旭日/发光点/曲形文本/顶刊复现…），仅 .R 共 ~617K。
- **`recipes_advanced_index.md`** —— 按技法分类导航 + "创新/合并思路示例"。出图想突破同质化、或想拼接
  多种技法（如云雨图+曲形文本、热图+边际折线+eyedroppeR 配色）时翻这里。
- 9 个 .rar 因无解压工具未附（主题见索引末尾），需要时去原库解压。
同样是技法参考、非发表级成品；最终出图按 SKILL.md 规范重做。
