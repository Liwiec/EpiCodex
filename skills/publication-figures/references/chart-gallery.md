# 图类型选型与开创思路画廊（chart gallery）

> 出图前先读本表"按意图选型"，再回 `SKILL.md` 落规范（尺寸/字体/配色/导出）。
> 本画廊提炼自用户私有 R 可视化库（50 套常用 + 130 套进阶"优雅"配方）。
> 用法：① 先按"数据情形→意图"定主图类型；② 看"优雅升级"有无更佳替代（避免千篇一律的柱/箱）；
> ③ 常用类型有现成代码 → `references/recipes_common_50/<类目>.R`（技法起点，**最终出图仍按 SKILL.md 规范重做**：
> 那些脚本用 base `pdf()`+硬编码 setwd+默认尺寸，不达发表级，仅供"该图怎么搭"参考）。
> ④ 进阶/优雅类型本库未附代码，按"关键包/技法"自行实现或检索。

**选型总原则**：先问"要表达什么关系"（比较 / 分布 / 相关 / 组成 / 趋势 / 流向 / 不确定性），
再选最朴素能讲清的图；只有当朴素图信息损失或同质化严重时，才升级到优雅变体。**新意服务表达，不为炫技**。

## 1. 组间比较（均值/中位数差异）
- **柱状图 + 误差线 + 显著性**：计数或均值少组对比 → `recipes_common_50/03显著性柱状图.R`、`04堆积百分比柱状图.R`、`05分类柱状图.R`。
- **箱线图（排序/差异/分面/多基因）**：连续变量分布对比，**优于柱图**（显示离散度）→ `06排序箱线图.R`/`07差异箱线图.R`/`08多基因差异箱线图.R`/`09多组差异箱线图.R`/`10箱线图分面.R`。
- **小提琴图（+内嵌箱/分面/多基因）**：样本量大、想看分布形状 → `11小提琴图.R`/`12多基因小提琴图.R`/`13小提琴图分面.R`。
- **成对差异分析**：`14成对差异分析.R`。
- **优雅升级**：
  - **云雨图 raincloud**（半小提琴+点+箱三合一，当代分布对比首选）→ `ggdist::stat_halfeye` / `gghalves`；进阶"半透明云雨""向云雨图加曲线文本"见进阶库。
  - **棒棒糖图 lollipop** 替代细高柱（更轻、少墨水）→ `30棒棒糖图.R` + 进阶"分布棒棒糖""线圈棒棒糖"。
  - **误差线点图 / 点+box 组合**（Nature 风）替代实心柱 → 进阶"组合展示误差线点图与柱状图""误差线组图加组间显著性"。
  - **偏差图 diverging** 正负对比 → `16偏差图.R` + 进阶"正负分布条形图"。
  - **别致/pro 版柱状图、砖块风 ggbrick、局部放大柱、金字塔图、蝴蝶图 ggpol** 见进阶库。

## 2. 分布（单变量/多组分布形状）
- **直方图 / 密度图**：单变量分布。
- **概率密度分布（按组）**：进阶"各大洲人类发展指数概率密度图"。
- **优雅升级**：
  - **山脊图 ridgeline / joyplot**（多组分布叠错峰，时间或类别维度极佳）→ `ggridges`；进阶"山脊图进阶版"。
  - **sina 图 / 环状 sina**：抖动点显真实分布 → `ggforce::geom_sina`；进阶"环状 sina"。
  - **三角密度图 ggsimplex**：三组分比例分布。
  - **月亮散点 gggibbous**：用月相编码第二比例变量。

## 16. 配色 / 工具
- **eyedroppeR**：从任意论文截图精准吸取配色十六进制（复刻顶刊配色神器）。

- 配色顺位仍以 SKILL §4 为准（ggsci lancet/nejm/jco → viridis/Okabe-Ito）；用 eyedroppeR 复刻时也要过色盲友好校验。
